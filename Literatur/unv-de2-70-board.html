DE2-70 Development and Education Board

Home > Training > University Program > Dev. & Education Boards > DE2-70

The Altera® DE2-70 Development and Education board is a modified version of the DE2 board with a larger FPGA and more memory. It is an excellent vehicle for learning about digital logic, computer organization, and FPGAs. Featuring an Altera Cyclone® II 2C70 FPGA, the DE2 board is designed for university and college laboratory use. It is suitable for a wide range of exercises in courses on digital logic and computer organization, from simple tasks that illustrate fundamental concepts to advanced designs.

DE2-70 Development and Education Board

The DE2-70 box includes:

  • The 8 x 6 inch DE2-70 board with a Cyclone II EP2C70 (896-pin package) FPGA
  • 12V AC/DC adaptor
  • USB cable
  • Plexiglas cover for the DE2-70 board
  • Installation guide
DE2-70 Board Information
Feature Description
FPGA
  • Cyclone II EP2C70F896C6 with EPCS64 64-Mbit serial configuration device
I/O Interfaces
  • Built-in USB-Blaster for FPGA configuration
  • Line In/Out, Microphone In (24-bit Audio CODEC)
  • Video Out (VGA 10-bit DAC)
  • Video In (NTSC/PAL/Multi-format) (2 ports)
  • RS232
  • Infrared port
  • PS/2 mouse or keyboard port
  • 10/100 Ethernet
  • USB 2.0 (type A and type B)
  • Expansion headers (two 40-pin headers)
Memory
  • 64 MB SDRAM, 2 MB SSRAM, 8 MB Flash
  • SD memory card slot
Displays
  • Eight 7-segment displays
  • 16 x 2 LCD display
Switches and LEDs
  • 18 toggle switches
  • 18 red LEDs
  • 9 green LEDs
  • Four debounced pushbutton switches
Clocks
  • 50 MHz clock
  • 27 MHz clock
  • External SMA clock input
